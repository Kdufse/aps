快速开始

只需简单几步即可在您的设备上安装 APatch Plus

1

准备工作

确保您的设备已解锁 Bootloader，并已安装自定义 Recovery（如 TWRP）。备份重要数据以防意外。

# 检查设备连接
adb devices
2

下载安装包

从 GitHub Releases 页面下载适用于最新版本 APatch Plus 安装包。

# 下载最新版本
wget https://github.com/Kdufse/APatch-Plus/releases/latest/download/APatchPlus_111925_1.6.1_on_main-release.apk
3

刷入设备

通过Recovery安装APatchPlus或使用 Fastboot 刷入 Boot 镜像。

# 通过Recovery安装APatchPlus
adb sideload APatch-Plus.apk
# 刷入Boot镜像 fastboot flash boot new-boot.img
#如果为Powershell，则为.\fastboot.exe flash boot .\new-boot.img
4

安装管理器

安装 APatch Plus 管理器应用，以便管理和配置模块。

# 安装管理器应用
adb install APatch-Plus.apk

# 启动应用并授权
